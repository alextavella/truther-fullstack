# Backend

## Running

```bash
pnpm dev
```

Open [http://localhost:3333/docs/](http://localhost:3333/docs/) with your browser to see the result.

## Drizzle

### Generate migrations:

```bash
pnpm drizzle-kit generate --name <name>
```

### Running migrations:

```bash
pnpm drizzle-kit migrate
```

## Seed

```bash
pnpm run seed
```

### Studio

```bash
pnpm drizzle-kit studio
```
